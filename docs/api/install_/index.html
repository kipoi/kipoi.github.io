install_model_requirements

install_model_requirements(model, source='kipoi', and_dataloaders=True)

Install model dependencies

Arguments

  • model (str): model name
  • source (str): model source

  • and_dataloaders (bool): if True, install also the dependencies for the default dataloader

  • .. deprecated:: 0.6.8 This will be removed in 0.7.0. installing packages in a running python env is error prone. Use command line interface of kipoi to install packages.
